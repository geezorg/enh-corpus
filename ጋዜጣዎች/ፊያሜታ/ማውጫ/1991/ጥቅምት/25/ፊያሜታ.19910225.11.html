አዲስ የትግልና የአንድነት ምዕራፍ

ከኢትዮጵያ ነፃ ኘነሬስ ጋዜጠኞች ማኀበር / ኢነጋማ/ አስተባባሪ ኮሚቴ የተሰጠ መግለጫ

ጥቅምት 28 / 1991 ( ኦችቶበር 30/1998 )

ከተቋቋመ 6 ዓመታት ያስቆጠረው የኢትዮጵያ ነፃ ኘሬስ ጋዜጠኞች ማኀበር /ኢነጋማ/ በየወቅቱ በሚደቀኑበት ማስናክሎችና ፈታኝ ሁነቶች ምክንያት ዓላማውን ሙሉ ሉሙሉ ከግብ ማድረስ አየተሳነው ዛሬ ከሚገኝበሪት ታሪካዊ ወቅት ላይ ይገኛል።

ከሁሉም በላይ በየዓመቱ የበርካታ አባላቱ ለእስራት ወከባ እንግልትና ለስደት መዳረግ ለእንቅስቃሴው መገታትና ለውጥኖች አለመሳካት የየበኩላቸውን አፍራሽ አስተዋፅዖ በተዋለ። በተለይ የዜጐችን በነፃ የመደራጀት ሕገመንግሥታዊ መብት ባለማክበር መንግሥት ለማኀበሩ ሕጋዊ ዕውቅና አለመስጠቱ የነፃው ኘሬስ አባላት የመሰባሰቢያና የመማማሪያ መድረካቸውን አጠናክረው መብታቸውን በማስከበር በሕግና ሕጋዊነት ከለላ ሙያዊ ግዴታቸውን በሚገባ ለመወጣትእንዳይችሉ መሰናክል ሆኖባቸዋል። ምንም እንኳ ተፅእኖ ቢደራረብባቸውም ደባና ሴራ ያላንበረከካቸው የኢነጋማ አባላት እንደትናቱ ሁሉ ዛሬም ለሰላም ለዴሞክራሲና ለሕግ የበላይነት ከትግሉ የመጀመሪያ ረድፍ በመሰለፍ በሕዝባቸው ወሰን የለሽ ድጋፍ እየታገዙ ትግሉን ቀጥለዋል።

የምንጊዜም መተማመኛችን መኩርያ መመኪያችን ማንም ሣይሆን ታሪክ ሰሪው የኢትዮጵያ ሕዝብ የመሆኑን ያህል ለሰው ልጆች ነንነትና እኩልነት እንዲሁ0ም ለሰብአዊ መብት ከበሬታ የሚሟገቱ የብሄራዊና ዓለም አቀፋዊ የዴሞክራሲያዊ ኃይሎች ከጐናችን መሰለፍ እናደንቃለን።

በተለይም ዓለም አቀፍ የጋዜጠኞች ፌዴሬሽን - ብራሰልስ ( እፍጅ ) ዓለም አቀፍ የሰበብአዊ መብት ተቋም ለንደን ( ኣምነስትይ እንተርናቲኦናል -ኣል ) ድንበር የለሽ የጋዜጠኞች መብት ተንከባካቢው ተቋም - ፓሪስ ( ረፖርተርስ ጻን ፍሮንቲአርስ - ርጽፍ ) ፣ የጋዜጠኞች መብት ተሟጋች ኮሚቴ - ኒውዮርክ ( ጮምሚትተአ ቶ ጵሮተችት ጆኡርናሊስትስ - ጭጵጅ ) ፣ ሂዩማን ራይትስ ዎች - ዋሽንግተን ( ሑማን ሪግህትስ ዋትችህ - ሕሯ ) ፣ እና ሌሎችም አህጉራዊና ዓለም አቀፋዊ ድርጅቶች ብሶታችንን በማስተጋባትና የኢህአዴግ መንግሥት በነፃው ኘሬስ ላይ ጫና ከማሳደር እንዲቆጠብ ፣ ጋዜጠኞችን ከእስር እንዲፈታ ፣ ለሙያ ማኀበሩ ሕጋዊ ዕውቅና እንዲሰጥ ፣ የኘሬስ ሕጉ እንዲሻሻልና የማኀበሩን የአመራር አባላት በስበብ አስባብ ከማዋከብ ፣ ከመሰርና በወንጀለኛነት እየፈረጀ ከማጉላላት ይልቅ ፣ ተቀራርቦ በመሥራት የሃገሪቱን የኘሬስ ነፃነት በተጨባጭ እውን በማድረግ ግዴታውን እንዲወጣ በመምክር ፣ በመማፀንና አቤቱታ በማሰማት የወሰዷቸውን ገንቢ እርምጃዎች ሁሉ በአድናቆት እንዘክራለን።

በተለይም ኢነጋማን ጨምሮ ከመቶ በላይ ሃገሮች በአባልነት የሚገኙበት ዓለም አቀፍ የጋዜጠኞች ፌዴሬሽን ( እፍጅ ) ከሜይ 3-7/1998 ባካሄደው 23ኛው ጉባኤው የኢትዮጵያ ነፃ ኘሬስ ጋዜዘጠኞችንና የማኀበራቸው ኢነጋማን ሁኔታ በአቢይ አጀንዳነት ይዞ በሰፊው ከተነጋገረ በኋላ ውሳኔዎችን ማሳለፉ ፣ የኢትዮጵያ ነፃ ኘሬስ አባላት ሕግንና ፣ ሙያዊ ግዴታን ተመርኩዘን ለምናከናውነው የጋዜጠኛነት ተግባር ወሰን የሌለው ኃይልና ብርታት እንደሚያጉናፅፈን በእርግጠኛነት መናገር እንችላለን።

ዓለም አቀፍ የጋዜጠኞች ፌዴሬሽን 23ኛ ጠቅላላ ጉባኤ በኢትዮጵያ ላይ ያሳረፈውን ትኩረት መሠረት በማድረግ የአይ.ኤፍ.ጄ ሴክሬታሪያት የሱየመው ሦስት አባላት የሚገኙበት የልዑካ ቡድን ከጥቅምት 7-15 ቀን 1998 ዓ.ም ( ኦችቶበር 17 -25 /1998) ወደ ሃገራችን በመምጣት የነፃውን ኘሬስ ተጨባጭ ሁኔታ በመመርመር በቂ ግንዛቤ ከማግኘቱም በላይ ከጠ/ሚ/ር መለስ ዜናዊና ከፍትህ ሚኒስትር አቶ ወረደ ወልድ ወልዱ ፣ ከማስታወቂያ ሚኒስትር ከአቶ ወ/ሚካኤል ጫሞና ከሌሎች የመንግሥት አካላት ጋርም ተወያይቷል።

በካሊን ማኬንዚ ዓለም አቀፍ የጋዜጠኞች ፌዴሬሽን ( ሑማን ሪግህትስ ኦፍፊቸር አንድ ወብ ጺተ ማናገር ) የተመራውና ሚስተር ፒተር ጐፍ ከኢንተርናሽናል ኘሬስ ኢኒስቲትዩት ፣ ( ጵረስስ ፍረአዶም ርዱኢሶር ፣ ሐአድ ፣ ኤዲቶሪአል ዸፓርትመንት ) እንዲሁም ሚስተር ማርቲን ኦቾሊ በዓለም አቀፍ ጋዜጠኞች ፌዴሬሽን የምሥራቅ አስተባሪ የተገኙበት የኸው የልዑካን ቡድን በአንድ ሣምንት የኢትዮጵያ ቆይታው አማራጭ ኃይሎች ምክር ቤት ፣ ከኢሰመጉ ፣ ከሕግ ባለሙያዎችና አግባብ ካላቸው ሌሎች ተቋማት ኃላፊዎች ጋር ተገናኝቶ ዲሞክራሲያዋና ሰብአዊ መብቶችን በሚመለከቱ ወሣኝ ጉዳዮች ላይ ተወይቷል።

የአይ.ኤፍ.ጄ የልዑካን ቡድን ከኢነጋማ አስተባባሪ ኮሚቴ ጋር ባካሄደው ሰፊና ጥልቅ ውይይት ማኀበሩን በማጠናከሩ ረገሉ ሊወሰዱ በሚገባቸው መሠረታዊ ነጥቦች ላይ ነተጋግሮ በጋራ ውሳኔዎችንም አሣልፏል። የኢነጋማን ሁለተኛ ጉባኤ ለመጥራት የማኀበሩን ጽ/ቤት በማደራጀት ረገድ ከዓለም አቀፍ የጋዜጠኞች ፌዴሬሽን ( እፍጅ) ስለሚስፈልገው የማቴሪያል ድጋፍ ፣ የአባላትን የሙያ ብቃት ለማሳደግ በሃገር ውስጥና በውጭ ሃገር ስለሚዘጋጁ የትምርት ኘሮግራሞች ፣ ኢነጋማ ስለሚኖረው የሕግ አማዛሪና ተግባራቱ ፣ ለኢንፎርሜሽን ልውውጥ የሚረዱ መሣሪያዎች በእርዳታ ስለሚገኙበት ሁኔታ ፣ የሙያውን ክቡርነት ለማጐልበትና ሕዝባዊ ተዓማኒነትን ለማጠናከር በማኀበሩ በኩለ ወሰድ ስለሚገባቸው አፋጣኝ እርምጃዎች በሃገራችን ዲሞክራሲያዊ ሥርዓትን ለማስፈን በመንግሥትና በነፃው ኘሬስ በኩል ሊኖር ስለሚገባው ጤናማ ግንኙነት ፣ በእስር ላይ ስለሚገኙ ጋዜጠኞች ፣ በስደት ውጭ ሃገር ስለሚገኙ የነፃው ኘሬስ አባላት ፣ ኢነጋማ በየዓመቱ ለአይ .ኤፍ.ጂ ( እፍጅ ) ማከፈል ስለሚገባው መዋጮ ፣ ለኢነጋማ አባላት ከአይ.ኤፍ.ጂ ስለሚሰጠው ኢንተርናሽናል ኘሬስ ካርድ በአጠቃላይ በቀጥታና በተዘዋዋሪ ከነፃው ሪሬስ አባላትና ሕልውና ጋር በተያያዙ በርካታ ነጥቦች ላይ ሰፊና ጥልቅ ውይይቶች ተደርገዋል።

ለኢነጋማና ጥንካሬና እድገት ታላቅ አስተዋፅዖ የሚያበርክቱና በአዲስ ምዕራፍነት የሚጠቀሱ አበረታች ስምምነቶች ላይ ተደርሷል። ተጨባጭ ውጤቶቹም በቅርቡ ይፋ እንደሚሆኑም እርግጠኞች ነን። ኢነጋማ ዓለም አቀፋ የጋዜጠኞች ፌዴሬሽን ( እፍጅ ) እና ኢንተርናሽናል ኘሬስ ኢኒስቲትዩት ( እጵእ ) የወሰዷቸውን መልካም የድጋፍና ትብብር እርምጃዎች ፣ በእቅድ የያዛቸውን ተስፋ ሰጪ መርሃ ግብሮች ባድናቆት እያስተወሰ ለተግባራዊ ትርጓማሜው የበኩሉን ተሳትፎ ያረጋግጣል።

በዚህ አጋጣሚ በተጋረጡበት መስናክለሎች ከእንቅስቃሴ ተገትቶ የቆየው የኢነጋማ አስተባባሪ ኮሚቴ ከዓለም አቀፉ የጋዜጠኞች ማኀበር ባገኘው የሞራልና በቅርብ ጊዜም ከእጁ አንደሚገባ ቃል በተገባለት የማቴሪያል ድጋፍና ተስፋ እየተበረታሩ በሙሉ ኃይሉ ለሚያከናውናቸው ተግባራት ከሁሉም በላይ የሚተማመንበትና የሚመካበት የኢትዮጵያ ሕዝብ የነፃ ኘፄረስ አባላት ፣ የዲሞክራሲያዉ ኃይሎችና በመላው ዓለም የሚገኙ አቻ ማኀበራት ያልተቆጠበ ትብብር ተጠናክሮ ይቀጥል ዘንድ ጥሪውን ያቀርባል።

ዓለም አቀፍ የጋዜጠኞች ፌዴሬሽን ( እፍጅ ) ልሁካን በኢትዮጵያ ያደረጉትን የሥራ ጉብኝት አስመልክተው ያጠናቀሩት ሪፖርት እንደደረሰው ኢነጋማ ለአባቱ ይፋ ያደርጋል። ከዚህም በተጨማሪ የሃገሪቱን ወቅታዊ ሁኔታና የነፃ ኘሬስን ኒና እንዲሁመ ከልዑካን ቡድኑ ጋር የተደረጉ ስምምነቶችን ካካተቱ ዐቢይ ጉዳዮች ላይ መወያየት የሚያስችል የአባላት ስብሰባ በቅርቡ እንደሚካሂድ ኢነጋማ ከወዲሁ ይገልፃል።

የኢትዮጰያ ነኀ ኘሬስ ጋዜጠኞች ማኀበር /ኢነጋማ / አስተባባሪ ኮሚቴ
